A very simple method to mask your affiliate links.


== Description ==
wpAFFI – Easy Affiliate Links Protection


= At a Glance =
* Very fast code execution
* Declare your links directly in the post
* No user interface required: Install, done!

= Antispam Bee counter =
`
<a href="[wpaffi link="http://affiliate.link"]">My Link</a>
`
Please adjust the anchor text accordingly.

== Installation ==
1. Download *wpAffi* plugin
1. Unzip the archive
1. Upload the folder *wpaffi* into *../wp-content/plugins/*
1. Go to tab *Plugins*
1. Activate *wpAffi*
1. Edit settings
1. Ready
